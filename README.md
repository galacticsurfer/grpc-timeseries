#Nifty daily close timeseries using gRPC python

CSV used from:
https://www1.nseindia.com/products/content/equities/indices/historical_index_data.htm

prerequisites:

***docker***

***docker-compose***

Steps to run:
***
***sudo docker-compose up --build***
***
